= listWebAuthnDevices

[.signature]
[subs="normal"]
----
AppDelegate.reachfive().listWebAuthnDevices(<<authToken>>)
----

== Description

When authenticated, the user can list the registred FIDO2 devices.

[NOTE]
====
* The WebAuthn feature must be enabled on your {company} account.
====
