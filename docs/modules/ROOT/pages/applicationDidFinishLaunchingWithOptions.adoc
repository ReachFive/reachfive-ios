= application(_:didFinishLaunchingWithOptions:)

[.signature]
[subs="normal"]
----
AppDelegate.reachfive().application(<<application, application>>, didFinishLaunchingWithOptions: <<applicationLaunchOptions, launchOptions>>)
----

== Description

Call this method in response to the equivalent UIKit method `application(_:didFinishLaunchingWithOptions:)`.

Allows ReachFive to initialize the SDK, by fetching information from the server about the client (default scopes, enabled features) and initializing the configured providers, if any.
