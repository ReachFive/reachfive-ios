== Description

`{docname}` is an array of objects that defines the specific social login providers (e.g., Google, Facebook, Apple) available in our iOS SDK.
You should define and configure these providers when you xref:index.adoc#initialize-ios-sdk[initialize the iOS SDK].

As of *{localdate}*, the only parameter you can pass to the objects is `variant` as shown on this page and described in <<Parameters, parameters>> below.
If no `variant:` is passed, the object takes the first variant containing `ios` in its name.
