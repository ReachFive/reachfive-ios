= signup

[.signature]
[subs="normal"]
----
AppDelegate.reachfive().signup(
    <<ProfileSignupRequest, profile>>,
    // Optional arguments
    <<redirectUrl>>,
    <<scope>>
)
----

== Description

Create and authenticate a new user with the specified data.

NOTE: Users can identify themselves with an email address or mobile number (if the `SMS` feature is enabled).

== Response

* *Type*: SignupFlow.<<AchievedLogin,AchievedLogin(authToken: AuthToken)>> | <<AwaitingIdentifierVerification>>
* *Error*: <<ReachFiveError>>

If the signup completes successfully, the promise returns either the authentication token (`AchievedLogin`) or indicates that the user must verify their identifier (`AwaitingIdentifierVerification`).  
Otherwise it is rejected and returns a `ReachFiveError`.
