= application(_:open:options)

[.signature]
[subs="normal"]
----
AppDelegate.reachfive().application(<<application, application>>, open: <<applicationOpenUrl, url>>, options: <<applicationOptions, options>>)
----

== Description

Call this method in response to the equivalent UIKit method link:https://developer.apple.com/documentation/uikit/uiapplicationdelegate/application(_:open:options:)[`application(_:open:options)`].

Allows ReachFive to handle URL of the form `reachfive-clientId://`, and any configured provider to handle their own native links.
