= application(_:continue:restorationHandler:)

[.signature]
[subs="normal"]
----
AppDelegate.reachfive().application(<<application, application>>, continue: <<applicationUserActivity, userActivity>>, restorationHandler: <<applicationRestorationHandler, restorationHandler>>)
----

== Description

Call this method in response to the equivalent UIKit method link:https://developer.apple.com/documentation/uikit/uiapplicationdelegate/application(_:continue:restorationhandler:)[`application(_:continue:restorationHandler:)`].

Allows ReachFive to handle universal links. Currently only used by xref:index.adoc#wechat-connect[WeChat].
